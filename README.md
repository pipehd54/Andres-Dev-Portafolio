Un portafolio personal diseñado bajo la estética de **Documentación de API**. El objetivo visual es reflejar un enfoque analítico, ordenado y orientado a la arquitectura de software, destacando habilidades en el diseño de sistemas y desarrollo backend.

## ⚙️ Especificaciones Técnicas

* **Estructura:** HTML5 semántico.
* **Estilos:** CSS3 puro (Grid / Flexbox).
* **Interactividad:** JavaScript (ES6+).
* **Despliegue:** GitHub Pages.

## 🏗️ Decisiones de Diseño

El diseño se aleja de los portafolios web tradicionales orientados al diseño gráfico y adopta una estructura de información de **Clave-Valor**. Visualmente simula la respuesta de un endpoint, comunicando madurez técnica y capacidad para organizar información compleja (características esenciales para el desarrollo con APIs y SQL).

## 🚀 Instalación Local

Para ejecutar este proyecto en tu entorno local:

1. Clona el repositorio:
   git clone https://github.com/pipehd54/Andres-Dev-Portafolio.git

2. Navega al directorio:
   cd Andres-Dev-Portafolio

3. Abre el archivo `index.html` en tu navegador preferido o utiliza una extensión como Live Server en tu IDE.
